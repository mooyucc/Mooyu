MooLog · 核心亮点

AI 智能整理 + 隐私优先 + 灵活项目管理，三位一体

🧠 AI 智能整理

  • • 粘贴文本或上传图片（自动 OCR 识别中英文），一键生成 Markdown 简报
  • • 自动提取文档日期并按周归档，告别手动归类
  • • 侧边 AI 助手：本周概览、对比上周、提炼待办、风险阻塞、生成汇报摘要
  • • 不满意？直接让 AI 助手修改，并一键应用到素材中

🔒 隐私优先，本地存储

  • • 所有数据保存在你自己指定的 .mlg 工作区文件中
  • • API Key 使用系统钥匙串加密，不上传至任何第三方
  • • 图片 OCR 全在本地完成，不依赖网络
重要提醒：调用大模型时内容会离开本机，请勿上传敏感信息 —— 除此之外，一切由你保管。

📂 灵活的项目管理

  • • 一个工作区文件可包含多个项目，左侧侧边栏一键切换
  • • 支持 macOS 原生文件菜单（⌘N / ⌘O / ⌘S / ⌘⇧S / ⌘W）
  • • 简报按周次组织，历史素材一目了然

典型使用流程

像用 Word 一样自然，五步完成简报整理

1

新建或打开工作区

打开 .mlg 工作区文件，像打开 Word 文档一样

2

选择项目

侧边栏选择住宅 / 商业 / 任意自定义项目

3

录入素材

粘贴会议记录，或拖入一张白板照片

4

查看简报

自动整理好的 Markdown 简报按周显示，可直接阅读或复制

5

⌘S 保存

关闭走人 —— 下次打开继续整理

# SchoolData 目录使用说明

## 目录说明

此目录用于存放学校信息的CSV文件。系统会自动扫描此目录下的所有CSV文件并导入到数据库。

## CSV文件格式

CSV文件应包含以下列：
- 第1列：分类（如：学校概况、学制阶段等）
- 第2列：子类（如：学校名称、性质等）
- 第3列：值（对应的数据）

## 使用方法

### 方法1：手动批量导入

将CSV文件放入此目录后，运行：

```bash
npm run import-all
```

### 方法2：自动监控（推荐）

启动文件监控，当有新文件添加或修改时自动导入：

```bash
npm run watch-schools
```

监控程序会：
- 自动检测新添加的CSV文件
- 自动检测修改的CSV文件
- 自动导入到数据库
- 显示导入结果

### 方法3：通过API导入（需要管理员权限）

管理员可以通过API触发批量导入：

```bash
POST /api/admin/schools/import-all
Authorization: Bearer <admin_token>
```

## 注意事项

1. CSV文件必须包含"学校名称"字段，否则会被跳过
2. 如果学校已存在，会自动更新数据
3. 如果学校不存在，会创建新记录
4. "N/A"值会被正确处理（布尔字段为false，文本字段为空）

## 文件命名建议

建议使用学校名称作为文件名，例如：
- `上海民办尚德实验学校.csv`
- `上海浦东新区民办万科学校.csv`
